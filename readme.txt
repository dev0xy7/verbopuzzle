# 🇮🇹 VerboPuzzle

VerboPuzzle è un'app educativa per imparare e consolidare i tempi verbali italiani attraverso un'attività di riconoscimento.

## Obiettivi

- Riconoscere il modo verbale.
- Riconoscere il tempo verbale.
- Ricevere un feedback immediato.
- Ottenere un rapporto finale con consigli di studio.

## Livelli

- 🎯 A0–A2
- 🚀 B1–B2
- 👑 C1–C2

## Tecnologie

- HTML5
- CSS3
- JavaScript
- Progressive Web App (PWA)
